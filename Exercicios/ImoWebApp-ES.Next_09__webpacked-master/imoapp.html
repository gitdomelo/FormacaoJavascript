Versão distribuída da ImoApp

Esta versão é igual à anterior, mas o JavaScript no navegador foi comprimido por Webpack.

Webpack á uma app, também escrita em JavaScript, capaz de "transpilar" JavaScript existente em JavaScript mais conciso e, se desejado, mais difícil de interpretar e, em consequência, copiar.

// Para lançar a webapp, execute na consola, no diretório do ficheiro Executar.js: node Executar.js

Visite a homepage ou teste a app requisitando uma página não existente.

// Para invocar webpack, instala-a com npm e executa na mesma pasta da app: webpack --config webpack.config.cjs --no-colors

Nota que nesta versão, apenas o JavaScript utilizado pelo navegador é reduzido com WebPack.
